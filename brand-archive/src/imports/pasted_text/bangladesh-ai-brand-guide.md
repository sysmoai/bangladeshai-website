=== CRITICAL RULES ===

1. NEVER use the Bangladesh national emblem (Shapla/water lily) — illegal under 1972 Order
2. NEVER use any government ministry logo, seal, or ICT Division / a2i branding
3. NEVER imply government affiliation
4. NEVER use "Smart Bangladesh" or "Digital Bangladesh" as own tagline
5. NEVER use stock photos of robots, glowing brains, or matrix code
6. NEVER use gradient blob shapes or SaaS startup aesthetics
7. NEVER use ALL CAPS for headings
8. NEVER place green and red colors directly adjacent
9. ALWAYS include both English AND Bangla taglines on key assets
10. ALWAYS maintain government-grade institutional quality
11. ALWAYS ensure WCAG AA contrast ratios (4.5:1 minimum for text)
12. ALWAYS design mobile-first (375px primary target — $100 Android phone on 4G in Bangladesh)
